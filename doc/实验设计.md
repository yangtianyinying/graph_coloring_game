# 线上实验设计

## 技术

静态网页, 发布到 github pages, 结束后下载数据到用户本地

前端 html, css, jsPsych（无后端；GitHub Pages 静态托管）

无数据库，数据存储到用户本地

## 核心研究问题

在图着色推断任务中，关注被试图结构复杂度的影响

## 刺激与颜色空间

图 $G = (V, E)$

$V$ : 节点标号, 节点颜色分布
$E$ : 边集, 


# 主实验区

## 功能

放入刺激集, 根据刺激集生成 trial 和 block

# 刺激集生成区

## 功能

自定义刺激集

自定义 block 数, 自定义 block 中 trial 数

自定义图, 

两种图, 自定顺序图和自由顺序图


包括自定义节点数, 自定义节点与节点的连接, 自定义初始化节点, 自定义初始化节点的颜色分布, 自定义节点的可视化位置 (拖到画布上, 呈现的时候也出现在画布上)
