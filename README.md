# 图着色推断实验

纯静态前端（HTML / CSS / JavaScript），使用 jsPsych 7（通过 `importmap` 从 `esm.sh` 加载）。

## 本地预览

在仓库根目录用任意静态服务器打开 `docs/` 即可，例如：

```bash
python -m http.server 8080 --directory docs
```

浏览器访问 `http://127.0.0.1:8080/`。

## GitHub Pages

仓库 **Settings → Pages**：Source 选分支（如 `main`），文件夹选 **`/docs`**。部署后访问 `https://<用户名>.github.io/<仓库名>/`。

详见 [`doc/GITHUB_PAGES.md`](doc/GITHUB_PAGES.md)。
